What We Do

Girls Who Code is a non-profit organization founded to help bridge the gender gap in tech. Here at Mission, we take that responsibility seriously.

HTML CSS Webpages

We teach the basics on how to build a website with HTML and CSS so no matter the project, our girls will understand how to manipulate the languages.

Mobile Apps

Later in the year, girls will have the option to choose a website or app format for their final projects. If an app is chosen, the basics of XCode will be learnt.

UI Design

Design is extremely important in computer science. Via our own lectures and those by guest speakers, our girls will understand the importance of placement size and color when building a program.
